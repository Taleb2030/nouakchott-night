# Media files — drop yours here

Replace these files with your own (keep the same names):

| File | Purpose |
|------|---------|
| `logo.png` (best), or `.jpg` / `.webp` / `logo.svg` | Navbar, hero, footer, browser tab |
| `restaurant.jpg` | Main hero / about photo |
| `videos/video1.mp4` | Hero background or gallery video 1 |
| `videos/video2.mp4` | Gallery video 2 |
| `videos/video3.mp4` | Gallery video 3 |

Supported formats: `.mp4`, `.webm`, `.jpg`, `.jpeg`, `.png`, `.webp`, `.svg`

After adding files, refresh the site (`npm run dev`).
